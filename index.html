About this project

Ancient Archives is a front-end project that loads story content from JSON and renders it with a terminal-style interface, typewriter effects, and optional logic exhibits. Built with vanilla HTML, CSS, and JavaScript—no frameworks.

Tech: ES6+, Fetch API, Intersection Observer, modular JSON data.
